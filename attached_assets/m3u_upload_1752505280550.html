Como Usar
Passos
  1. Selecione um arquivo M3U do seu computador
  2. Clique em "Carregar e Visualizar"
  3. Navegue pelos filmes com interface visual
  4. Use filtros de busca e categoria
  5. Clique nos filmes para ver detalhes
Funcionalidades
  • Visual Interface com imagens dos filmes
  • Busca Pesquisa por nome do filme
  • Filtros Organização por categorias
  • Reprodução Links diretos para assistir
  • Export Gerar nova lista M3U
  • Download Baixar HTML navegável
Download do HTML
Como baixar o HTML navegável:
  1. Carregue seu arquivo M3U usando o formulário acima
  2. Na página do visualizador, clique no botão "Baixar HTML"
  3. O arquivo HTML será baixado automaticamente
  4. Abra o arquivo baixado no navegador para usar offline
Vantagens do HTML baixado:
  • Funciona offline (sem internet)
  • Contém todos os dados do M3U embutidos
  • Mantém todas as funcionalidades (busca, filtros, etc.)
  • Pode ser compartilhado facilmente
